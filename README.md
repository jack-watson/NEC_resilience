To run, download all files and run bosWashMain.m. 

Modify bw_fpath (bosWashMain.m, line 12) if BosWash_graph.mat file is not located in "data" subfolder of working directory. 

Dependencies include two visualization functions written by other users and published to MathWorks FileExchange. For full citations, see alluvialflow.m and fanChart.m in /plotting subfolder.
1. Alluvial flow diagram, Alexander Carmeli (2018): https://www.mathworks.com/matlabcentral/fileexchange/66746-alluvial-flow-diagram
2. Fan chart, Ameya Deoras (2016): https://www.mathworks.com/matlabcentral/fileexchange/48006-fanchart-visualize-percentiles-of-time-series-data
